Read `brief/creative-brief.md` and `brief/tools-and-models.md` first. If either is missing, stop and tell the user to run `/onboard`.

Then draft a script appropriate to the format:

- **Narration-driven**: VO lines + on-screen direction in brackets
- **Dialogue**: character names + lines, with shot direction
- **Silent / visual**: shot-by-shot visual beat list with timing

Match the target runtime in the brief (rough rule: ~150 words/min for VO).

Save as `scripts/drafts/draft-NN.md` (increment NN if drafts already exist). Include a header with: target runtime, draft number, date, and a one-line revision summary.

After writing, summarise the draft in 3–5 bullets and ask whether to:
- iterate (write another draft),
- promote (copy to `scripts/final/script.md`),
- proceed to `/storyboard`.
